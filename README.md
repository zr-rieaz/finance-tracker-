# Taka Tracker 💰

Personal finance tracker for family expense management.

## Build APK via GitHub Actions

1. Push this repo to GitHub
2. Go to **Actions** tab
3. Click **Build Android APK** → **Run workflow**
4. Wait ~5-7 minutes
5. Download `TakaTracker-APK` from the Artifacts section

## Setup (optional - for Gemini AI features)

Add `VITE_GEMINI_API_KEY` in GitHub repo Settings → Secrets → Actions.

## Local Development

```bash
npm install
npm run dev
```
